# معرفی پروژه

این پروژه یک **سامانه مدیریت گالری/فروشگاه ساعت** است که با هدف پیاده‌سازی یک ساختار کامل **Frontend + Backend** توسعه داده شده است.
در این سامانه، کاربران می‌توانند وارد سیستم شوند و بسته به نقش خود، به بخش‌های مختلف دسترسی داشته باشند.

### قابلیت‌های کلی پروژه

* احراز هویت کاربران با **JWT**
* مدیریت **محصولات**
* مدیریت **دسته‌بندی‌ها**
* مدیریت **کاربران**
* پنل مدیریتی نقش‌محور
* ویرایش پروفایل و تغییر رمز عبور
* ارتباط کامل بین Angular و Laravel از طریق API

# راه‌اندازی سریع

## 1) راه‌اندازی بک‌اند

وارد پوشه `backend` شوید و دستورات زیر را اجرا کنید:

```bash id="root-2"
composer install
php artisan key:generate
php artisan migrate --seed
php artisan serve
```

بک‌اند به صورت پیش‌فرض روی آدرس زیر اجرا می‌شود:

```text id="root-3"
http://127.0.0.1:8000
```

---

## 2) راه‌اندازی فرانت‌اند

وارد پوشه `frontend` شوید و دستورات زیر را اجرا کنید:

```bash id="root-4"
npm install
npm start
```

فرانت‌اند به صورت پیش‌فرض روی آدرس زیر اجرا می‌شود:

```text id="root-5"
http://localhost:4200
```

* فایل `backend/.env` نباید در مخزن قرار بگیرد و فقط `backend/.env.example` باید نگهداری شود.
